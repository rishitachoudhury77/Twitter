A full Twitter sentiment analysis workflow, including:

Importing libraries (pandas, numpy, matplotlib, seaborn, wordcloud, regex).

Loading the Sentiment140 dataset and renaming columns.

Mapping sentiment values (0 → negative, 2 → neutral, 4 → positive).

Text-cleaning function (removing URLs, mentions, hashtags, symbols).

Creating word clouds (positive / negative / neutral).

Sentiment counts and visualizations (bar charts, line charts).

Some topic frequency / word frequency calculations.

Overall: It is a notebook that loads the dataset, cleans the text, analyzes sentiment, and generates visualizations.
